🎯 Formula scor compus
scor = 0.6 × medBac + 0.2 × dif1 + 0.1 × dif2 + 0.1 × dif3
Candidatii sunt sortati descrescator dupa scor compus per facultate. Bugetistii (formFunding=1) ocupa intai locurile buget; locurile buget ramase merg la taxisti cu cele mai mari scoruri. Taxistii (formFunding=2) ocupa locurile taxa. Restul → WAITING_LIST. Serviciul comunica cu admissions-service prin Feign Client cu CircuitBreaker + Retry (Resilience4j).
